# The 12 Principles Worth Keeping

## 1. Kill the Gamer Aesthetic Immediately

This is universally agreed upon.

Remove:

* Dark mode
* Neon borders
* Glowing cards
* Cyberpunk colors
* Fancy gradients everywhere

Replace with:

```txt

bg-slate-50

Cards:
bg-white
shadow-sm
border-slate-200

Primary:
blue-600
blue-700

slate-900
slate-600
```

### Why?

Judges associate:

```txt

=
Student Project

Light Mode + White Space
=
Enterprise Software
```

This single change probably gives the biggest perceived quality improvement.

## 2. Keep the Sidebar Copilot

Several documents contradict each other.

Some suggest:

```txt
Embed AI deeply into workflow
```

Others say:

```txt
Keep sidebar
```

The correct answer for a hackathon is:

### KEEP THE SIDEBAR

Reason:

Embedded workflow AI means:

* React state explosion
* More bugs
* More UI work
* More testing

Sidebar Copilot already works.

Just make it prettier.

Think:

```txt
Coupa's Ask Cassie
```

not

```txt
Custom AGI Workflow Engine
```
